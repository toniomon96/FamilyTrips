# Plan Templates

Copy one of these outlines into a new file under `src/data/trips/` and fill in the typed `Trip` object. Keep sensitive details out of source if the link needs real privacy.

## Beach Trip

- `kind`: omit or use `'trip'`
- Core sections: stay, bookings, itinerary, people, contacts, checklist, packing, budget
- Helpful packing categories: Travel docs, Clothes, Toiletries, Beach / Pool, Baby, Tech, Dogs
- Copy blocks to consider: arrival instructions, grocery plan, beach day plan

## Road Trip

- `kind`: omit or use `'trip'`
- Core sections: itinerary, bookings, people, contacts, checklist, packing, budget
- Helpful checklist categories: Car, Route, Stay, Food, Kids, Dogs
- Helpful packing categories: Car / Road trip, Snacks, Tech, Clothes, Toiletries

## Cookout

- `kind`: `'event'`
- Core sections: schedule, food, supplies, eventTasks, people, contacts, budget, copyBlocks
- Helpful food categories: Grill, Sides, Drinks, Dessert
- Helpful tasks: shopping, yard setup, coolers, trash, cleanup
- Copy blocks to consider: invite text, what to bring, day-of reminder
